# 🦉 フクちゃん掲示板

## プログラミング学習者向け掲示板  サービス概要

学習中によく出る疑問やエラーをカテゴリごとに整理し、迷わず調べられる学習サポートアプリです。  
黒板風の画面から悩みを選ぶだけで、フクちゃんが原因や対処ポイントを分かりやすく案内してくれます。  
学習のつまずきを減らし、次の一歩へ進みやすくしてくれる“学習ナレッジベース × アドバイスツール”です。

---

## このサービスへの思い・作りたい理由

自分自身が学習中に何度もエラーや理解不足に苦しみ、
「今の状況に合ったアドバイスが欲しい」と感じる場面が多くありました。

プログラミング学習では AI に質問したりできますが、AI の回答が間違っていることもあり、  
「実際に同じ状況を経験した受講生のリアルなエピソードが最も役に立つ」 と強く思いました。

そこで、受講生たちが経験したつまずきや解決方法を整理し、  
後輩が同じ壁にぶつかったときにすぐ役立つ“学習ナレッジ集”として形にすれば、  
確かな価値が生まれると考えました。

フクロウの“フクちゃん”が学習者に寄り添い、
受講生の経験をもとにアドバイスを返してくれるアプリを目指して、  
このサービスを作ることにしました。

---

## 🎯 ユーザー層について

### RUNTEQ受講生
- よくあるつまずきを知りたい
- 人に質問する前に自力で解決したい

### 独学のプログラミング初学者
- 今の自分と似た悩みを探したい
- 調べ方が分からず詰まることが多い

**理由**
プログラミング学習者は「悩みをうまく言葉にできない」「調べ方が分からず詰まる」ことが多いです。  
こうした背景から、悩みをカテゴリからすぐ探せて、フクちゃんがやさしく案内してくれるこのサービスは、  
人に質問する前に気軽に使える“最初の相談相手”として役立ちます。

---

## 🖼️ サービスの利用イメージ

1. トップ画面の黒板に「エラー」「モチベ」「設計」などカテゴリが並ぶ
2. カテゴリを押すと “よくある悩みボタン” が表示される
3. ボタンを押すとフクちゃんからアドバイスが表示される
4. 受講生の知見は「アドバイス案募集」項目から追加され、
   採用されたものは新しい悩みボタンとして増えていく

  **学習者の経験がアプリ自体を成長させる仕組み**

## 📣 ユーザーの獲得について

- よく読まれた悩みランキングなどでリピート導線を作る
