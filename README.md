# 🦉 フクちゃん掲示板

## プログラミング学習者向け掲示板  サービス概要

学習中によく出る疑問やエラーをカテゴリごとに整理し、迷わず調べられる学習サポートアプリです。  
黒板風の画面から悩みを選ぶだけで、フクちゃんが原因や対処ポイントを分かりやすく案内してくれます。  
学習のつまずきを減らし、次の一歩へ進みやすくしてくれる“学習ナレッジベース × アドバイスツール”です。

---

## このサービスへの思い・作りたい理由

自分自身が学習中に何度もエラーや理解不足に苦しみ、
「今の状況に合ったアドバイスが欲しい」と感じる場面が多くありました。

RUNTEQでは AI に質問してヒントをもらえる環境がありますが、AI の回答が間違っていることもあり、  
「実際に同じ状況を経験した受講生のリアルなエピソードが最も役に立つ」 と強く感じました。

そこで、受講生たちが経験したつまずきや解決方法を整理し、“後輩が同じ壁にぶつかったとき、  
すぐに役立つ悩み辞典” を作れたら確実に価値になると思いました。

フクロウの“フクちゃん”が学習者に寄り添い、
受講生の経験をもとにアドバイスを返してくれるアプリを目指して、  
このサービスを作ろうと決めました。

---

## 🎯 ユーザー層について

### RUNTEQ受講生
- よくあるつまずきを知りたい
- Slackで質問する前に軽く解決したい

### 独学のプログラミング初学者
- 今の自分と似た悩みを探したい
- 調べ方が分からず詰まることが多い

**理由**
この層は「悩みの言語化が苦手」「調べ方が分からない」状態になりやすいため、
辞典形式とキャラによるアドバイスが特に刺さる。

---

## 🖼️ サービスの利用イメージ

1. トップ画面の黒板に「エラー」「モチベ」「設計」などカテゴリが並ぶ
2. カテゴリを押すと “よくある悩みボタン” が表示される
3. ボタンを押すとフクちゃんからアドバイスが表示される
4. 受講生の知見は「アドバイス案投稿」から追加され、
   採用されたものは新しい悩みボタンとして増えていく

  **学習者の経験がアプリ自体を成長させる仕組み**

## 📱 レスポンシブ対応

※すべてのデバイスに対応しているわけではありません。

## 🧰 セットアップ手順

```bash
# リポジトリをクローン
git clone https://github.com/fuku-0420/owl-forest-app.git
cd owl-forest-app
